Portal Full Dev - Desenvolvimento

O projeto foi inspirado a partir de uma necessidade do cotidiano dos colaboradores da equipe, com intuito de realizar testes dos envios de HSM para números de celulares específicos de diferentes ambientes como por exemplo Code7MessagingIntegration API desenvolvida por Core Dev e também diretamente pelo ambiente Flex da mensageria sem precisar abrir o CRM ou Portal Web para fazer simples testes de validação de ambiente.

O modelo de aplicação criada para esse projeto foi Angular com o ASP.NET Core , criando uma estrutura com duas aplicação web moderna e avançada, contendo um BackEnd com ASP.NET Core 5.0 robusto e seguindo os padrões de desenvolvimento de API da Code7, separadamente temos o FrontEnd com Angular/CLI a interface do usuário, trabalhando com a modularizações de funcionalidades para futuramente criar vários outros processos que devam ser otimizados em funcionalidades do portal. Esse modelo oferece uma hospedagem de ambas aplicações publicadas como uma unidade de deploy.
